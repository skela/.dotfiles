This file provides guidance to AI coding agents when working with code in this project.

## Project

This code is for my dotfiles, which contains configuration and scripts for my linux and macos computers.

## Coding style

When in doubt look at `editorconfig` for rules on coding style and formatting when generating code.

## Rules

- Do not read .env files ever
- Do not access the private folder ever.
- Do not commit code to git ever.
- Do not modify third-party libraries or packages source code ever.
- Do not disable or bypass security features ever.
- Do not hardcode secrets, tokens, or credentials in code.
