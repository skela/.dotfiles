# Acknowledgments

Hyprspace is built on the work of several open-source projects.

## Core

- [AeroSpace](https://github.com/nikitabobko/AeroSpace) by Nikita Bobko — the tiling window manager Hyprspace is forked from (MIT License)

## Companion Tools

- [SketchyBar](https://github.com/FelixKratz/SketchyBar) by Felix Kratz — status bar used for workspace indicators
- [JankyBorders](https://github.com/FelixKratz/JankyBorders) by Felix Kratz — window border highlights

## Bundled Dependencies

- [HotKey](https://github.com/soffes/HotKey) — global keyboard shortcut handling (MIT)
- [TOMLKit](https://github.com/LebJe/TOMLKit) — Swift TOML config parsing (MIT)
- [tomlplusplus](https://github.com/marzer/tomlplusplus) — underlying TOML C++ parser (MIT)
- [ANTLR v4](https://github.com/antlr/antlr4) — shell-like language parser (BSD-3-Clause)
- [swift-collections](https://github.com/apple/swift-collections) — advanced Swift collection types (Apache 2.0)
- [ISSoundAdditions](https://github.com/InerziaSoft/ISSoundAdditions) — system volume control (MIT)

Full license texts are available in the `legal/` directory of each release artifact.
